How to Use Test Pages

When using the accessibility scanner in the application:

  1. Enter test-sample or test-accessible in the URL field
  2. Click "Scan" to run the accessibility check
  3. The system will use the local test pages instead of attempting to connect to external websites

Why Test Pages?

These test pages are useful when:

  • Network access is restricted in the deployment environment
  • External domain access is limited
  • You want to test the scanner functionality without internet connectivity
  • You want predictable and consistent test results
